Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполните контентом достаточным,\чтобы по высоте он начинал захватывать вторую статью,как показано в примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равный .2em, при помощи свойства margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равный .2em, при помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организовано по првому краю внутри данной статьи для этой картинки

Нижний блок, подвад. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
